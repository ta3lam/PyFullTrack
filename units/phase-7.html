🏆 Phase 7 — Real-World Projects

Build 4 complete, portfolio-worthy projects using everything you've learned.

📊 Project 1 — Sales & Business Analysis

Load a retail dataset (9,994 rows). Answer 15 business questions. Build interactive charts. Write an executive summary.

Key Questions: Which category generates the most profit? Which region underperforms? What is the seasonal trend?

🦠 Project 2 — COVID-19 Data Analysis

Analyze global COVID-19 data from Our World in Data. Compare countries, track waves, build world maps with Plotly.

Key Questions: Which countries had the fastest vaccine rollout? How did waves differ in severity?

⚽ Project 3 — Sports Analytics

FIFA player dataset (18,000+ players). Analyze skills by position, country performance, and value vs skill.

Key Questions: Is player market value correlated with actual skill? Which age group peaks?

💰 Project 4 — Finance & Stock Market

Historical stock prices (Apple, Google, Amazon). Moving averages, risk analysis, multi-stock comparison.

Key Questions: Which stock had the best 5-year return? What is the risk/return relationship?
