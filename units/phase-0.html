🚀 What is Python & Data Analysis?

Unit 1 — Set up your environment and write your first Python code. No experience needed!

What is Python?

Python is a programming language — a way to give instructions to a computer. It reads almost like English, which makes it perfect for beginners.

Python is used in: data analysis, machine learning, web development, finance, automation, and much more.

python
# This is Python code — # is a comment, Python ignores it print("Hello, World!") # Show text on screen print(2 + 2) # Math print("Data" * 3) # Repeat text
Output
Hello, World! 4 DataDataData

What is Data Analysis?

Data Analysis means examining data to find useful answers — like "Which product sells the most?" or "What month had the highest revenue?"

Python makes this easy with powerful libraries: Pandas, NumPy, and Matplotlib.

💡 Tip: You don't need to be a math expert. Python does the calculations — you just need to ask the right question.

Why Python for Data?

  • ✅ Free and open source
  • ✅ Used by Google, Netflix, NASA, banks, hospitals
  • ✅ Millions of developers worldwide
  • ✅ Thousands of ready-made libraries
  • ✅ Easiest language to learn and read

🚀 Installing Anaconda

Unit 1 — Set up your environment and write your first Python code. No experience needed!

What is Anaconda?

Anaconda is a Python distribution that comes with 300+ pre-installed data science libraries. It's the easiest way to get started — no manual installation needed.

📝 Note: Anaconda is free for personal use. Download from: anaconda.com/download

Installation Steps

  1. Go to anaconda.com/download
  2. Choose your OS (Windows / Mac / Linux)
  3. Download the installer and run it
  4. Click Next until installation is complete
  5. Open Anaconda Navigator from the Start menu

What's Included?

python
# These come pre-installed with Anaconda: import numpy as np # Fast math import pandas as pd # Data tables import matplotlib.pyplot as plt # Charts import seaborn as sns # Beautiful charts import scipy # Scientific computing import sklearn # Machine learning print("All ready! 🎉")
Output
All ready! 🎉
💡 Tip: After installation, open Anaconda Prompt and type: conda list to see all 300+ installed packages.

Keeping Packages Updated

python
# In Anaconda Prompt / Terminal: conda update --all # Update everything conda install package_name # Install new package pip install package_name # Alternative installer

🚀 Jupyter Notebook Basics

Unit 1 — Set up your environment and write your first Python code. No experience needed!

What is Jupyter Notebook?

Jupyter Notebook is an interactive coding environment that runs in your browser. You write code in cells and see results instantly below each cell — perfect for data analysis.

How to Open Jupyter

  1. Open Anaconda Navigator
  2. Click Launch under Jupyter Notebook
  3. Your browser opens at localhost:8888
  4. Click New → Python 3 (ipykernel)

Essential Keyboard Shortcuts

python
# ── Inside a cell (Edit Mode) ── Shift + Enter # Run cell, move to next Ctrl + Enter # Run cell, stay Alt + Enter # Run cell, insert new below # ── Outside a cell (Command Mode — press Esc) ── A # Add cell Above B # Add cell Below D D # Delete cell M # Change to Markdown (text) Y # Change to Code Z # Undo H # Show all shortcuts
💡 Tip: Press Esc then H to see all keyboard shortcuts — learning them saves hours!

Cell Types

Code cells — run Python code and show output.
Markdown cells — write formatted text, headings, and notes using simple symbols.

python
# This is a Code cell — it runs Python # Markdown cell example (change cell type with M): # # Big Heading # ## Medium Heading # **bold text** # *italic text* # - bullet point
📝 Note: Save your notebook with Ctrl+S. Jupyter also autosaves every few minutes.

🚀 Python in VS Code

Unit 1 — Set up your environment and write your first Python code. No experience needed!

Why VS Code?

VS Code is a free, powerful code editor by Microsoft. Many professionals prefer it over Jupyter for writing larger Python files and projects.

Setup VS Code for Python

  1. Download from code.visualstudio.com
  2. Install the Python extension by Microsoft
  3. Install the Jupyter extension to run notebooks
  4. Press Ctrl+Shift+P → "Select Python Interpreter" → choose your conda env

Essential VS Code Shortcuts

python
# Running code: F5 # Run with debugger Ctrl+F5 # Run without debugger Shift+Enter # Run selected line (Python extension) # Editing: Ctrl+/ # Toggle comment Alt+Up/Down # Move line up/down Ctrl+D # Select next occurrence Ctrl+Shift+K # Delete line # Navigation: Ctrl+P # Quick open file Ctrl+` # Open terminal Ctrl+Shift+P # Command palette (most important!)
💡 Tip: Use Ctrl+Shift+P and type what you want — VS Code has thousands of commands accessible this way.

Running Python Files

python
# Create a file: analysis.py # Add your code: import pandas as pd print("Hello from VS Code!") # Run in terminal: # python analysis.py # Or press the ▶ button in the top right
📝 Note: For data analysis work, Jupyter notebooks are better for exploring. Use .py files for reusable scripts and pipelines.
🎯 TASK Practice Exercise
In VS Code: create a file called 'hello.py', write a script that prints your name and today's date, then run it from the terminal.
Starter code:
python
# hello.py # Import the date module from datetime import ??? name = "Your Name" today = ???.today() print(f"Hello, {name}!") print(f"Today is: {today}")
✅ Solution
python

Output
Hello, Ahmed! Today is: 2024-01-15 Year: 2024
